Prediction System

Players


Description

To use the player prediction system. You have to enter five players names and a map. This is done by using the drop down menus.
Don't worry. The list is sorted alphabetically. You will only see players that have match history but the not on the map select.
The system will tell you if you choose a player name that does not work.

Priorities

The prediction system uses a priority based ranking system. The lower the number the higher the priority.
1. Player Ranking
2. ADR
3. KDA

Teams


Description

To use the team prediction system.You need to pick up teams and a map. All the team names and maps are provided in the select lists.
All the select lists are sorted alphabetically. If you had an incorrect select (e.x. same team names), the system will give you instructions in red text.
Due to the limit number of records in our database, we can't give some predictions on some teams and maps. If the system gives you this warning, please try to change the selected team or map.

Visualization System

Players


Description

To use the player visualization system. You first must enter a event name done by a drop down. This list is alphabetically sorted.
After that, you have to enter in a match id. There is no other way to find a match, you will just have to guess and check until you find the one you want.
Finally, you can then select a map for that match and you will be able to see the visualized data.
You will be able to see KDA and ADR for all ten players for that map/match.

Teams


Description

To use the team visualization system. You can pick a team from the drop down menu and click submit button.
You will see all the maps that the picked team have joined in the x-axis label.
The bar chart will display all the winning rate of the selected team.
